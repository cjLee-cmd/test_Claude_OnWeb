Claude Code를 활용한 웹 개발 테스트 프로젝트입니다.

## 프로젝트 구조

```
.
├── src/          # 소스 코드
├── docs/         # 문서
├── tests/        # 테스트 코드
├── examples/     # 예제 코드
└── README.md     # 프로젝트 설명
```

## 시작하기

### 필요 사항

- Node.js (권장: v16 이상)
- Git

### 설치

```bash
# 레포지토리 클론
git clone https://github.com/cjLee-cmd/test_Claude_OnWeb.git
cd test_Claude_OnWeb

# 의존성 설치 (프로젝트 타입에 따라 다름)
npm install
```
