# 시작 가이드

## 개요

test_Claude_OnWeb 프로젝트에 오신 것을 환영합니다.

## 개발 환경 설정

### 1. 사전 요구사항

- Node.js v16 이상
- Git
- 코드 에디터 (VS Code 권장)

### 2. 프로젝트 클론

```bash
git clone https://github.com/cjLee-cmd/test_Claude_OnWeb.git
cd test_Claude_OnWeb
```

### 3. 의존성 설치

```bash
npm install
```

### 4. 프로젝트 실행

```bash
npm start
```
